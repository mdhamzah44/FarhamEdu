About FarhamEdu

FarhamEdu is an online learning platform developed to help students access educational content easily. This website provides courses, video lessons, quizzes, and now also includes live classes for better interaction and understanding.

The platform is designed with a simple and clean user interface so that students can learn without any confusion and focus on their studies.

Project Information

This project "FarhamEdu" has been developed as a BCA Final Year Project. The main objective of this project is to create an online learning platform where students can access study materials, video lectures, quizzes, and live classes.

The website is designed using HTML ,CSS , JavaScript and the Python flask and socket.io for the backend and peer to peer connections for live classes and mongoDB Atlas for the Database and cloudinary for the storage and Vercel for the serverless functions and render for the web services
